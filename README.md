## Install Nerd fonts (our update)
[fonts](https://gist.github.com/davidteren/898f2dcccd42d9f8680ec69a3a5d350e)

## Install lazygit
```console
brew install lazygit
```

## Install Rust Analyzer
In order to get rust lsp tools
```console
brew install rust-analyzer
```
